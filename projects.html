Work Projects

I've worked on, and have been working on several projects while at Cox Automotive:

  • I'm currently working on a Chatbot called 'Ask Kelley' that utilizes Natural Language Parsing, and will hopefully be debuting on KBB.com soon. Will add a link and more details when public though.
  • I did a massive migration of my team's flagship service away from an old infrastructure stack of terraform code, deployed using Jenkins, onto Elastic Beanstalk, and onto AWS CDK, GitHub Actions, and AWS Fargate. This migration took several months of steady work, but led to reduced overhead costs of ~20%, and even reduced our API response times by ~10% as well.
  • Another big cost reduction project I did was consolidating different caching services we were using across our projects, into a single AWS MemoryDB cache. Aside from reducing our infrastructure complexity immensely, MemoryDB is saving my company approximately $9k/month compared to the other databases and caching tools we were using previously.
  • One of the more visible projects I did was related to the 'My Wallet' tool on Autotrader.com and Kelley Blue Book. I was tasked with creating the Database for storing information related to this project, as well as the API service that the front-end uses to allow user interactions with it. You can read about MyWallet in this Autotrader Article
  • A big projects that's always ongoing with my team is expanding the notification suite at Autotrader and Kelley Blue Book. I work on stuff like saved car searches, tracking price drops for selected vehicles, and various 're-engagement' emails for consumers browsing different sections of the sites.
